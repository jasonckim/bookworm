## Functionality 

This app returns a list of 10 books based on the search term you input. Info returned:\
-`Thumbnail Image`\
-`Title`\
-`Author`\
-`Published Date`\
-`Description (Toggle: On/Off)`

In addition, you can sort the books by publish date, from `Oldest` to `Newest`.\

Happy searching!
